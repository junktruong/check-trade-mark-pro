Precheck Sheet Errors API (for Frontend)

Endpoint:
POST /api/precheck/sheet/errors

Purpose:
- Chỉ trả về các lỗi (WARN/BLOCK) từ Warning Words, Block Words, TMHunt và Gemini Policy.
- Không dùng cơ chế continue.
- Không trả về các row PASS.
- Gemini Policy trả giải thích tiếng Việt.
- Giữ nguyên endpoint cũ /api/precheck/sheet để các luồng hiện tại tiếp tục sử dụng.
- API này nhận link Google Sheet.

Endpoint:
POST /api/precheck/sheet/errors

Request Body:
{
  "sheetUrl": "https://docs.google.com/spreadsheets/d/.../edit#gid=0",
  "options": {
    "enableTextCheck": true,
    "enableTmCheck": true,
    "enablePolicyCheck": true
  }
}
